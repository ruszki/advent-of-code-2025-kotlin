# Advent of code 2025

Solutions for [Advent of code 2025](https://adventofcode.com/2025) in Kotlin.

All code here is public domain. No restrictions.

Naming convention is \<day\>a.kt or \<day\>b.kt, depending on whether the first task (a), or the second (b). If there are multiple solutions then \<day\>a-\<solution id\>.kt or \<day\>b-\<solution id\>.kt.
